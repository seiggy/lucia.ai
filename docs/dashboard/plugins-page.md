# Plugins

The Plugins page lets you extend Lucia's functionality through a plugin ecosystem. It is organized into three tabs.

![Plugins](/img/dashboard/plugins.png)

## Installed

The **Installed** tab lists all plugins currently installed on your Lucia instance. For each plugin you can:

- **Enable or disable** -- toggle a plugin on or off without uninstalling it.
- **Uninstall** -- remove the plugin and its data entirely.

Plugin version and description are shown alongside each entry.

## Store

The **Store** tab lets you browse available plugins from configured repositories. Each plugin listing shows its name, description, author, and version. Click **Install** to add a plugin with one click. Lucia handles downloading, registration, and activation automatically.

## Repositories

The **Repositories** tab manages the sources that populate the plugin store. You can:

- **Add** a repository by providing its URL.
- **Remove** a repository to stop listing its plugins.
- **Sync** a repository to refresh its plugin catalog.

By default, the official Lucia plugin repository is pre-configured.
