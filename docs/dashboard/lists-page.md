# Lists

The Lists page lets you manage todo and reminder lists that Lucia uses to track items on your behalf.

![Lists](/img/dashboard/lists.png)

## Todo Lists

Create and organize todo lists for different purposes (grocery shopping, home projects, etc.). Each list contains items that can be checked off, reordered, or deleted. Items can be added manually from the dashboard or created by Lucia during conversations.

## Reminder Lists

Reminder lists work like todo lists but include a date and time for each item. When a reminder's scheduled time arrives, Lucia will notify you through your configured notification channels.

## Managing Lists

- **Create** a new list by clicking **New List** and choosing the list type.
- **Rename** or **delete** lists from the list header menu.
- **Add items** using the input field at the bottom of each list.
- **Reorder items** by dragging them into the desired position.

Lists are synced in real time, so changes made through voice commands or other interfaces appear immediately on the dashboard.
