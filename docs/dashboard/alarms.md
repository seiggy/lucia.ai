# Alarms

The Alarms page lets you create and manage scheduled alarms with advanced audio controls.

![Alarms](/img/dashboard/alarms.png)

## Scheduling

Alarms use CRON expressions for flexible scheduling. Set one-time alarms or recurring patterns (weekdays only, every morning, etc.). The next trigger time is displayed alongside each alarm.

## Volume Ramping

Configure alarms to start quietly and gradually increase volume over a specified duration. This provides a gentler wake-up experience compared to a sudden full-volume alarm.

## Voice Control

When an alarm fires, it can be dismissed or snoozed using voice commands through Lucia. Say "stop" or "dismiss" to silence the alarm, or "snooze" to delay it by a configurable interval.

## Sound Library

Choose from built-in alarm sounds or upload your own audio files. Uploaded files are stored in the sound library and can be reused across multiple alarms. Supported formats include MP3, WAV, and OGG.

## Managing Alarms

The alarm list shows each alarm's name, schedule, sound, and enabled state. Toggle alarms on or off without deleting them.
